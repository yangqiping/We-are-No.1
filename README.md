## 初探星辰

### 项目简介

初探星辰是一款将节日和节气、诗词结合的一个文化科普小程序。可以帮助大家更多的了解和学习诗词文化和节气等文化。



### 项目成员

杨麒平、范美茹、赵芸卉、薛子玥、李星



### 项目背景

社会越来越现代化，传统农耕文明遗留下来的节气文化和古人遗留下来的诗词名篇与其产生的文化也很少有人愿意学习与了解。也有一些喜欢传统文化的人没有一个公共的平台供他们交流。



### 项目功能

- 供大家了解节气、节日和诗词的知识；

- 自定义制作贺卡、卡片；
- 分享自己所写的诗词；
- 浏览他人所写诗词并点赞评论等；
